Register File

Ok great! We can perform our operations, but rather than just manually setting inputs and discarding outputs, how do we save them in memory? The register file is the name for the construct holding data for each register. The number of registers, as well as their number of bits, can vary, but Minecraft tends to work well with 8-bit registers in terms of compact builds.
